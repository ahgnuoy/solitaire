### Build

```
npm run build
node app.js
```

### APIs

#### Set Game

- url
    - /game
- method
    - POST
- body
    - game data(board set)
- return
    - null

#### Get Solution

- url
    - /solution
- method
    - GET
- return
    - solution Array [pair]
